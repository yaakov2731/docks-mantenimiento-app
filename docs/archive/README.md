# Archive

Material archivado para referencia histórica.

## Estructura

- `legacy-root/`: notas y documentos que antes estaban en la raíz del repo.
- `legacy-bot-local/`: integración local antigua del bot, fuera del runtime actual en Railway.
- `legacy-panel/`: panel estático anterior, fuera de la app React actual.
- `superpowers/`: planes y specs históricos de implementación.

## Regla

Nada de esta carpeta debe considerarse fuente de verdad del runtime actual salvo revisión explícita.
