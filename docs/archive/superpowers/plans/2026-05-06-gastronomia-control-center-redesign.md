# Gastronomía Control Center Redesign — Plan

## Objetivo

Ejecutar rediseño visual y de orden del módulo Gastronomía sin cambiar lógica de negocio.

## Pasos

1. Crear navegación interna compartida del módulo.
2. Extender estilos gastronómicos compartidos en `client/src/index.css`.
3. Integrar navegación compartida en `Index`, `Planificación`, `Confirmaciones` y `Asistencia`.
4. Migrar `Personal` y `Liquidación` a `DashboardLayout` y lenguaje visual gastronómico.
5. Ajustar `Planificación` para que el flujo operativo quede más explícito.
6. Correr `npm run build`.
7. Desplegar a Railway.
8. Validar respuesta básica del servicio.
